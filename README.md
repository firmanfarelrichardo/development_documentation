# 🚀 VCC Documentation Repository

Repository ini digunakan untuk menyimpan seluruh dokumentasi, file tugas, serta video demonstrasi mata kuliah **Virtual Cloud Computing (VCC)**.

Di dalam repository ini terdapat implementasi deployment server, konfigurasi layanan cloud, virtualisasi, networking, dan demonstrasi hosting berbasis Linux Server.
